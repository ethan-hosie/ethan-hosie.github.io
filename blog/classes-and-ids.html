Classes and ids

What is a class?

A class is a selector that allows a developer to select a group of elements using the same class name and apply styles to it. The class selector is written with a period followed by the class name.

What is an id?

An id is used to select a single element within the code and need to be unique to each element. The id selector is written with a # followed by the id name.

Which should you use?

You can use whichever one you want as they do the same thing, but you should use id's for specific elements that you want to apply a certian style to.
